## Trabalho Final da disciplina Programação Orientada a Objetos 1
Este projeto acadêmico, desenvolvido em 2019, trata-se de um sistema em JAVA para controle de produtos. Com interface gráfica simples, o sistema simula a venda de produtos cadastrados, além de permitir exclusões, alterações e buscas.

--------------
### Análise Atual do Projeto
Após este projeto novas disciplinas foram cursadas, como Programação Orientada a Objetos 2 e Engenharia de Software, evidenciando que o sistema desenvolido há época peca em vários pontos, dentre eles:

- Não segue padrões de desenvolvimento, como o MVC;
- Sistema de login é inseguro, uma vez que o usuário e senha estão dispostos no código;
- Não há integração com banco de dados;
- A auto-inserção de dados nos campos de atualização, colhendo dados automáticamente da tabela, em "alterar produto";
